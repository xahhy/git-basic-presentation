Git 存储二进制文件过程

  • 需要存储的内容「content」
  • 需要存储的文件类型「blob」
  • 「类型+内容」计算SHA-1值, 根据SHA-1值生成文件路径和名称
  • 使用zlib生成压缩文件并存储 
							
											.git/objects/7d/e504fb32f2227f3616463b55fd02501ce2dd32

Git 存储文件类型

blob 二进制文件

tree 文件树 (目录)

commit 提交记录

tag 标签

小总结👌

  1. 打包每一个二进制文件并存储: blob 💎
  2. 为当前目录结构生成一份快照: 文件树对象 tree 📁
  3. 为当前快照标记一段有意义的文字: 提交 commit 📦
